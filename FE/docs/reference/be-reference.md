# BE 참고 추출

원본 문서: `최종_LG_Able_Band_개발산출물_260609.docx`

이 문서는 FE 개발 중 백엔드 담당자와 API 계약을 맞추기 위해 참고할 내용만 정리한다.
FE 담당자는 `BE/` 코드를 직접 수정하지 않는다.

## 백엔드 역할

- Spring Boot API Server
- 인증 관리
- 사용자/기기/이벤트/알림/보호자/긴급 요청 관리
- 외부 시스템 이벤트 수신
- 이벤트 정규화
- 알림 이력 저장
- UWB 위치 안내 세션 관리

## BE 내부 처리로 명시된 기능

| FR ID | 기능 | 설명 |
| --- | --- | --- |
| FR-010 | 이벤트 정규화 | ThinQ, 센서, 웨어러블 등 외부 시스템에서 들어오는 서로 다른 이벤트 형식을 서비스 내부의 공통 이벤트 형식으로 변환한다. |
| FR-011 | 이벤트 분류 요청 | 정규화된 이벤트를 룰 엔진 또는 ML 서버를 통해 생활 알림/위험 알림/긴급 알림 등으로 분류한다. |
| FR-029 | 생활 이벤트 제공 인터페이스 | 외부 연동 시스템이 생활 이벤트를 서비스로 전달할 수 있는 수신 인터페이스를 제공한다. |
| FR-030 | 위험 이벤트 감지 인터페이스 | 외부 연동 시스템이 위험 이벤트를 서비스로 전달할 수 있는 수신 인터페이스를 제공한다. |
| FR-031 | 위치 정보 제공 인터페이스 | 외부 연동 시스템이 UWB 위치 또는 거리 정보를 서비스로 전달할 수 있는 인터페이스를 제공한다. |

## FE가 백엔드에 요청해야 할 API 계약

- 인증 응답 형식
- 사용자 접근성 프로필 조회/저장 형식
- 기기 목록과 연결 상태 형식
- 알림 목록/상세 응답 형식
- 알림 확인/다시 듣기 처리 방식
- 보호자 등록/조회/수정 형식
- 긴급 요청 생성과 전달 상태 형식
- 이벤트 이력 조회 필터 형식
- UWB 탐색 세션 시작/갱신/종료 형식
- 실패 응답 형식

## DB 객체 참고

| 객체 | 설명 |
| --- | --- |
| USER | 서비스 사용자 계정 및 접근성 프로필 |
| GUARDIAN | 보호자 계정 및 연락 정보 |
| USER_GUARDIAN | 사용자-보호자 연결 관계 |
| DEVICE | 연동 가전/센서/웨어러블/위치 안내 대상 기기 |
| DEVICE_EVENT | 생활/위험/위치 관련 원천 이벤트 |
| ALERT | 사용자 또는 보호자에게 발송되는 알림 |
| ALERT_DELIVERY | 알림 채널별 전송 및 확인 결과 |
| EMERGENCY_REQUEST | 긴급 도움 요청 |
| UWB_NAVIGATION_SESSION | UWB 가전 위치 안내 세션 및 거리 이력 요약 |
| USER_FEEDBACK | 알림 확인, 다시 듣기, 응답 기록 |

## 주요 필드 참고

### USER

| 필드 | 의미 |
| --- | --- |
| USER_ID | 사용자 식별자 |
| NAME | 사용자명 |
| ACCESSIBILITY_TYPE | 시각장애인/청각장애인 장애 유형 |
| NOTIFICATION_PREFS | 음성/진동/화면/텍스트 설정 |

### DEVICE

| 필드 | 의미 |
| --- | --- |
| DEVICE_ID | 기기 식별자 |
| USER_ID | 소유 사용자 |
| DEVICE_TYPE | 세탁기/센서/TV/UWB 등 |
| LOCATION_SUPPORTED | UWB 안내 가능 여부 |

### DEVICE_EVENT

| 필드 | 의미 |
| --- | --- |
| EVENT_ID | 이벤트 식별자 |
| DEVICE_ID | 발생 기기 |
| EVENT_TYPE | 생활/위험/위치 |
| NORMALIZED_PAYLOAD | 공통 이벤트 포맷 |

### ALERT

| 필드 | 의미 |
| --- | --- |
| ALERT_ID | 알림 식별자 |
| USER_ID | 알림 대상 |
| EVENT_ID | 관련 이벤트 |
| ALERT_TYPE | 생활/위험/긴급/위치 |

### UWB_NAVIGATION_SESSION

| 필드 | 의미 |
| --- | --- |
| SESSION_ID | 위치 안내 세션 |
| USER_ID | 요청 사용자 |
| TARGET_DEVICE_ID | 찾기 대상 가전 |
| LAST_DISTANCE_M | 마지막 거리 |
| STATUS | 진행/도착/실패/취소 |
