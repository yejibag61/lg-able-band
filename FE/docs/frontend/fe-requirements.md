## FE에서 참고만 할 내부 기능

| FR ID | 기능 | 담당 | FE 참고 방식 |
| --- | --- | --- | --- |
| FR-008 | 생활 이벤트 수집 | BE/외부 연동 | 수집 결과가 알림으로 내려온다고 가정 |
| FR-009 | 위험 이벤트 수집 | BE/외부 연동 | 위험 알림 데이터 구조 참고 |
| FR-010 | 이벤트 정규화 | BE 내부 | FE는 정규화된 이벤트/알림만 받음 |
| FR-011 | 이벤트 분류 요청 | BE/ML 내부 | FE는 분류 결과와 긴급도만 표시 |
| FR-029 | 생활 이벤트 제공 인터페이스 | BE API | FE 구현 지시 아님, API 계약 참고 |
| FR-030 | 위험 이벤트 감지 인터페이스 | BE API | FE 구현 지시 아님, API 계약 참고 |
| FR-031 | 위치 정보 제공 인터페이스 | BE API/UWB | FE는 거리/상태 표시만 담당 |

## 비기능 요구사항 중 FE 반영 항목

| NFR ID | 구분 | FE 반영 |
| --- | --- | --- |
| NFR-001 | 성능 | 위험/긴급 알림은 즉시 인지되는 UI로 표현한다. |
| NFR-002 | 접근성 | 음성, 진동, 화면, 텍스트 방식으로 핵심 정보를 인지할 수 있게 한다. |
| NFR-003 | 보안/개인정보 | 실제 토큰/비밀번호/민감정보를 화면 코드나 커밋에 넣지 않는다. |
| NFR-004 | 안정성 | 외부 API 실패, 조회 실패, 권한 없음 안내를 제공한다. |
| NFR-005 | 가용성 | MVP 시연 중 핵심 화면에 접근 가능하게 mock fallback을 준비한다. |
| NFR-008 | 사용성 | 핵심 기능은 3단계 이내 접근, 큰 버튼, 명확한 문구, 고대비 UI를 적용한다. |
| NFR-009 | 위치 안내 정확성/안전성 | UWB 신뢰도 낮음, 신호 실패, 미지원 상태를 명확히 알린다. |

## MVP와 후속 개선 구분

| 구분 | MVP 포함 | 후속 개선 |
| --- | --- | --- |
| 음성 안내 | 기본 음성 안내, 알림 다시 듣기 | 음성 속도/크기 조절, 안내 문구 커스터마이징 |
| 알림 방식 | 접근성 프로필 기반 음성/진동/화면/텍스트/색상 보조 알림 | 상황별 고급 패턴 추천, 사용자 학습 기반 자동 최적화 |
| UWB 위치 안내 | 세탁기 찾기, 거리별 음성/진동, 도착 안내 | 복수 가전 동시 탐색, 방향 안내 고도화 |
| 가전 기능 | 세탁 완료, 냉장고 문 열림, 공기질, 위험 신호 | 리모컨 찾기, 식재료 찾기, AI 생활패턴 분석 |
