# FE 사용자 흐름

원본 문서: `최종_LG_Able_Band_개발산출물_260609.docx`

이 문서는 원본 서비스 흐름도를 FE 화면 전환과 사용자 행동 중심으로 재정리한 것이다.

## 기본 탭 진입 기준

`FE/app`은 LG ThinQ 레퍼런스를 참고해 하단 탭 기반으로 구성한다.
각 흐름은 사용자가 어느 탭에서 시작하는지 먼저 정하고, 세부 화면으로 이동한다.

| 탭 | 사용자가 하는 일 | 연결 흐름 |
| --- | --- | --- |
| 홈 | 오늘 상태 확인, 긴급 요청, 최근 알림 확인 | 긴급 도움 요청, 알림 상세, UWB 위치 안내 |
| 알림 | 생활/위험/긴급 알림 확인 | 생활 신호 알림, 위험 신호 알림, 다시 듣기, 이벤트 이력 |
| 기기 | 기기 연결과 UWB 위치 안내 | 기기 연동, UWB 가전 위치 안내 |
| 메뉴 | 접근성 설정과 보호자 연결 관리 | 접근성 프로필 설정, 보호자 연결 |

## 생활 신호 알림 흐름

1. 외부 시스템이 생활 이벤트를 제공한다.
2. BE/ML이 이벤트를 생활 알림으로 분류한다.
3. FE는 사용자 접근성 설정에 맞는 알림 표현을 표시한다.
4. 사용자는 실시간 알림 화면 또는 웨어러블 화면에서 알림을 확인한다.
5. 사용자는 알림 상세로 이동하거나 다시 듣기를 실행한다.
6. 확인 상태는 이후 이력 화면에서 조회 가능해야 한다.

FE 필요 화면:

- 홈 화면
- 실시간 알림 화면
- 알림 상세/다시 듣기 화면
- 이벤트 이력 화면
- 웨어러블 알림 화면

## 위험 신호 알림 및 보호자 연결 흐름

1. 외부 시스템이 위험 이벤트를 감지한다.
2. BE/ML이 위험 알림 또는 긴급 알림으로 분류한다.
3. FE는 위험도를 강하게 표시한다.
4. 사용자가 응답하지 않거나 위험 등급이 높으면 보호자 알림 흐름이 이어진다.
5. 보호자는 보호자 위험 알림 화면에서 상황을 확인한다.
6. 보호자는 확인, 연락, 이력 보기를 수행할 수 있어야 한다.

FE 필요 화면:

- 실시간 알림 화면
- 보호자 위험 알림 화면
- 이벤트 이력 화면
- 웨어러블 위험/긴급 알림 화면

## 긴급 도움 요청 흐름

1. 사용자가 홈 화면 또는 웨어러블 화면에서 긴급 도움 요청을 실행한다.
2. FE는 요청 전후 상태를 명확히 보여준다.
3. BE는 보호자에게 요청을 전달하고 이력을 저장한다.
4. FE는 요청 성공, 발송 실패, 보호자 없음 같은 상태를 표시한다.
5. 요청 이력은 이벤트 이력 화면에서 확인할 수 있어야 한다.

FE 필요 화면:

- 홈 화면
- 웨어러블 긴급 알림/응답 화면
- 보호자 위험 알림 화면
- 이벤트 이력 화면

## 알림 다시 듣기 흐름

1. 사용자가 실시간 알림 또는 알림 이력에서 알림을 선택한다.
2. 알림 상세/다시 듣기 화면으로 이동한다.
3. 사용자가 다시 듣기를 누른다.
4. FE는 재생 중, 재생 완료, 실패 상태를 보여준다.
5. 사용자가 확인 완료를 누르면 알림 응답 상태가 변경된다.

FE 필요 화면:

- 실시간 알림 화면
- 알림 상세/다시 듣기 화면
- 이벤트 이력 화면
- 웨어러블 다시 듣기 화면

## UWB 가전 위치 안내 흐름

1. 사용자가 가전 위치 안내 메뉴로 이동한다.
2. 위치 안내 대상 가전을 선택한다.
3. FE는 UWB 지원 여부와 탐색 가능 상태를 보여준다.
4. 탐색 시작 후 거리, 근접도, 진동 상태, 음성 안내 문구를 표시한다.
5. 도착 시 도착 상태와 긴 진동 2회를 표시한다.
6. 신호 실패, 미지원, 권한 없음 상태를 안내한다.

FE 필요 화면:

- UWB 가전 위치 안내 화면
- 웨어러블 UWB 거리/진동 안내 요약 화면

## UWB 거리별 안내 정책

| 상황 | 밴드 음성 안내 |
| --- | --- |
| 멀리 있음 | “세탁기까지 약 4미터입니다.” |
| 가까워지는 중 | “세탁기까지 약 2미터입니다. 가까워지고 있습니다.” |
| 거의 도착 | “세탁기 근처입니다.” |
| 도착 | “세탁기 앞입니다.” |

| 거리 | 진동 |
| --- | --- |
| 3m 이상 | 느린 진동 |
| 1~3m | 중간 간격 진동 |
| 1m 이내 | 빠른 진동 |
| 도착 | 긴 진동 2회 |
