# FE/app 작업 지침

`FE/app`은 LG Able Band의 사용자 웹/모바일 메인 앱이다.
사용자와 보호자가 서비스를 관리하고 상태를 확인하는 큰 화면을 담당한다.

## 우선 참고 문서

- API 계약: `../../docs/api/final.md`
- FE 구현 순서와 mock/API 교체 기준: `../../docs/frontend/fe-final-api-guide.md`
- 화면 설계와 흐름: `../../docs/frontend/fe-screens.md`, `../../docs/frontend/screen-api-map.md`, `../../docs/frontend/fe-user-flows.md`
- 디자인 참고: `../../docs/frontend/fe-design-reference.md`, `../../docs/frontend/home-screen-plan.md`
- 보조 참고: `../../docs/api/api-spec-by-screen.md`, `../../docs/api/api-spec-draft.md`, `../../docs/reference/be-reference.md`, `../../docs/reference/ml-reference.md`

`../../docs/api/final.md`가 백엔드와 합의된 확정본이다. 앱 구현 중 이 파일은 수정하지 않는다.

## 담당 화면

- 로그인/역할 선택 화면
  - 사용자/보호자 역할 선택
  - 로그인 성공 후 역할별 첫 화면 이동
  - 로그인 실패 안내
- 홈 화면
  - 오늘의 상태 요약
  - 최근 생활/위험 알림
  - 연동 기기 상태
  - 긴급 도움 요청 버튼
- 접근성 프로필 설정 화면
  - 시각장애인/청각장애인 유형 선택
  - 음성, 진동, 화면, 텍스트, 색상 보조 알림 설정
  - 기본값 적용과 저장 실패 안내
- 기기 연동 화면
  - LG ThinQ, 센서, 웨어러블 연동 기기 목록
  - 연결 상태 표시
  - Mock 기기 연결/해제 흐름
- 실시간 알림 화면
  - 생활/위험/긴급 알림 카드
  - 위험도, 발생 위치, 발생 시간 표시
  - 확인, 다시 듣기, 상세 보기 액션
- 알림 상세/다시 듣기 화면
  - 상세 문구, 발생 시간, 기기명 표시
  - 다시 듣기와 확인 완료 액션
  - TTS 실패 또는 데이터 없음 안내
- 보호자 연결/보호자 위험 알림 화면
  - 보호자 등록과 연결 상태 관리
  - 보호자에게 전달된 위험 알림 확인
  - 연락, 확인, 이력 보기 액션
- UWB 가전 위치 안내 화면
  - 위치 안내 대상 가전 선택
  - 거리, 진동 상태, 음성 안내 문구 표시
  - 탐색 시작/중지, 다시 안내 액션
- 이벤트/알림 이력 화면
  - 기간 필터
  - 이벤트 목록과 상세 보기
  - 다시 듣기 액션

## 추천 구현 순서

1. 로그인과 사용자/보호자 역할 분기
2. 공통 API 클라이언트와 mock data 구조
3. 공통 레이아웃과 하단 탭 구조
4. 홈 화면
5. 실시간 알림, 알림 상세, 확인, 다시 듣기
6. 접근성 설정
7. 기기 연동과 보호자 연결
8. 이벤트/알림 이력
9. UWB 위치 안내

## 구현 원칙

- `FE/app`은 관리, 설정, 조회, 보호자 연결 같은 전체 서비스 흐름을 담당한다.
- LG ThinQ 레퍼런스의 밝은 배경, 큰 제목, 둥근 카드, 하단 탭 구조를 참고하되 Able Band의 접근성/알림 중심 목적에 맞게 재구성한다.
- 백엔드 API가 없으면 실제 API 응답 형태를 예상한 mock data로 화면을 먼저 완성한다.
- 화면은 사용자가 3단계 이내로 핵심 기능에 접근할 수 있게 구성한다.
- 접근성 요구사항을 우선하여 큰 터치 영역, 고대비, 명확한 상태 문구를 사용한다.
- 웨어러블에서 즉시 표시해야 하는 간단 알림 UI는 `FE/wearable`에 구현한다.
- 화면별로 로딩, 에러, 빈 데이터, 권한 없음, 외부 API 오류 상태를 준비한다.
- `src/App.jsx`, `src/App.css`, `src/index.css`가 현재 주요 진입점이며, 기능이 커지면 컴포넌트와 mock data를 분리한다.
